## 🎯 Topics Covered

### 🔹 What is Salesforce?

Salesforce is a **cloud-based Customer Relationship Management (CRM) platform** that helps businesses manage customers, sales, marketing, and support in one place.

It solves problems like:

* Scattered customer data
* Poor communication between teams
* Missed sales opportunities
* Manual and repetitive tasks

Salesforce provides a **single platform** to store customer data, automate workflows, and improve business productivity.

---

### 🔹 What is CRM?

CRM stands for **Customer Relationship Management**.

It helps companies:

* Manage leads and customers
* Track communication (calls, emails, meetings)
* Improve customer satisfaction
* Increase revenue

👉 Simple meaning: A system to manage customer relationships digitally.

---

### 🔹 Salesforce Data Model Basics

#### 🧱 Objects

Objects are like **database tables** that store information.

Examples:

* Standard Objects → Lead, Account, Contact
* Custom Objects → Offer (created in hands-on)

#### 🧾 Records

A record is a **single row of data** in an object.

Example: One customer entry inside the Contact object.

#### 🏷️ Fields

Fields are **columns** that store specific data.

Examples:

* Text
* Number
* Currency
* Date

## 👩‍💻 Admin vs Developer

| Salesforce Admin         | Salesforce Developer               |
| ------------------------ | ---------------------------------- |
| Works using clicks       | Works using code                   |
| Creates objects & fields | Builds apps & integrations         |
| Automates workflows      | Writes Apex & Lightning components |
| Maintains system         | Extends functionality              |
